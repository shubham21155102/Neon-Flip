# Assets Directory

This directory should contain the following assets:

## Images
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

## Sounds
- `jump.mp3` - Sound effect for gravity flip
- `crash.mp3` - Sound effect for collision/death

## Instructions

1. Create or obtain the assets listed above
2. Place them in this directory
3. Update `app.json` if you change filenames

## Asset Sources

You can create these assets using:
- **Icons**: [Canva](https://www.canva.com), [Figma](https://www.figma.com)
- **Sounds**: [Freesound](https://freesound.org), [Zapsplat](https://www.zapsplat.com)
- **Generate with AI**: DALL-E, Midjourney for images
